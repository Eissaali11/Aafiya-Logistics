## 📌 نظرة عامة

**Aafiya Logistics** هو موقع ويب تعريفي احترافي لشركة متخصصة في خدمات **توظيف القوى العاملة** وإدارة الموارد البشرية في المملكة العربية السعودية. يتميز الموقع بتصميم عصري متجاوب مع دعم كامل لجميع الأجهزة، ويشمل لوحة تحكم إدارية محمية.

```

║                                                      ║

╚══════════════════════════════════════════════════════╝
```

## 🚀 طريقة التشغيل

### ▶️ الطريقة الأولى — فتح مباشر
```bash
# 1. نسخ المستودع
git clone https://github.com/Eissaali11/Aafiya-Logistics.git
cd Aafiya-Logistics

# 2. فتح الصفحة الرئيسية
open index.html   # macOS
start index.html  # Windows
```

### ▶️ الطريقة الثانية — سيرفر محلي (موصى به)
```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx serve .

# ثم افتح: http://localhost:8000
```
